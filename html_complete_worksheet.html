Q01
🐾 내가 좋아하는 동물 소개 페이지
저장 파일명 → animal.html
<h1><h2><p><br><hr> <strong><em><mark> <div><span><ul><img> + <header><main><footer>
20
점
📋 문제 상황
내가 가장 좋아하는 동물을 소개하는 페이지를 만들어요. 기존 조건을 완성한 후 시맨틱 태그로 페이지 구조를 감싸세요.
📌 기존 조건
  • 1<h1> 로 동물 이름을 대제목으로 작성하세요
  • 2<h2> 로 "특징", "사는 곳", "먹이" 소제목 3개를 작성하세요
  • 3<div> 로 각 소제목 영역을 묶으세요
  • 4<img> 로 동물 이미지를 넣으세요 (alt 속성 필수)
  • 5<ul> + <li> 로 특징 3가지 이상 목록으로 작성하세요
  • 6<strong>, <em>, <mark> 를 각각 1번 이상 사용하세요
  • 7<span> 을 사용해서 동물의 색깔이나 크기 부분을 감싸세요
  • 8소제목 사이에 <hr> 구분선을 넣으세요
✨ 추가 조건 — 시맨틱 태그로 페이지 구조 잡기
  • 추가1전체 내용을 <header>, <main>, <footer> 로 나누세요
  • 추가2<header> 안에 <h1> 제목과 <img> 이미지를 넣으세요
  • 추가3<main> 안에 특징, 사는 곳, 먹이 영역을 모두 넣으세요
  • 추가4<footer> 안에 <p> 로 작성자 이름과 날짜를 넣으세요

Q02
📚 나의 독서 목록 페이지
저장 파일명 → book.html
<h1><h2><ul><ol><li> <strong><del><mark><a><img> + <header><main><aside><footer>
20
점
📋 문제 상황
나의 독서 목록 페이지를 만들어요. 기존 조건을 완성한 후 aside 를 추가해서 추천 도서관 정보를 사이드바로 넣으세요.
📌 기존 조건
  • 1<h1> 로 "나의 독서 목록 📚" 제목을 작성하세요
  • 2<h2> 로 "읽고 싶은 책", "이미 읽은 책" 소제목 2개를 작성하세요
  • 3<div> 로 두 영역을 각각 묶으세요
  • 4<ol> 로 읽고 싶은 책 3권 이상 작성하고 1순위 책에 <mark> 와 <strong> 을 사용하세요
  • 5<ul> 로 읽은 책 2권 이상 작성하고 <del> 취소선을 적용하세요
  • 6<img> 로 책 표지 이미지를 1개 이상 넣으세요 (alt 속성 필수)
  • 7<a> 로 책 검색 사이트 링크를 새 탭으로 연결하세요
✨ 추가 조건 — 시맨틱 태그로 구조 잡기 + aside 사이드바 추가
  • 추가1전체 페이지를 <header>, <main>, <footer> 로 나누세요
  • 추가2<header> 안에 <h1> 제목을 넣으세요
  • 추가3<main> 안에 읽고 싶은 책 / 읽은 책 영역을 모두 넣으세요
  • 추가4<aside> 를 <main> 아래에 추가하고, 안에 "📍 추천 도서관" 소제목과 도서관 이름 목록 (<ul>) 을 넣으세요
  • 추가5<footer> 안에 작성자 이름과 날짜를 넣으세요

Q03
🎬 영화 리뷰 페이지
저장 파일명 → movie.html
<h1><h2><h3><p><ul><hr> <strong><em><small><span><img><a> + <article><section> + <form><input><textarea><button>
20
점
📋 문제 상황
영화 리뷰 페이지를 만들어요. 기존 조건을 완성한 후 article 로 리뷰 영역을 감싸고 한줄 리뷰를 남기는 간단한 폼을 추가하세요.
📌 기존 조건
  • 1<h1> 로 영화 제목, <h2> 로 "기본정보" "줄거리" "나의 평점" 소제목 3개를 작성하세요
  • 2<div> 로 영화 정보 카드 전체를 묶으세요
  • 3<img> 로 영화 포스터 이미지를 넣으세요 (alt 속성 필수)
  • 4<ul> + <li> 로 감독, 장르, 개봉년도를 목록으로 작성하세요
  • 5줄거리 <p> 에서 <strong>, <em> 을 사용하세요
  • 6평점 별 이모지에 <span> 을 사용하세요
  • 7<small> 로 출처를 표시하고, <a> 로 예매 링크를 새 탭으로 연결하세요
  • 8소제목 사이에 <hr> 구분선을 넣으세요
✨ 추가 조건 — article / section + 한줄 리뷰 폼 추가
  • 추가1영화 정보 카드 전체를 <article> 태그로 감싸세요
  • 추가2기본정보 / 줄거리 / 평점 각 영역을 <section> 태그로 나누세요
  • 추가3<hr> 아래에 "✏️ 한줄 리뷰 남기기" <h2> 소제목을 추가하세요
  • 추가4<form> 안에 이름 입력 <input type="text">, 별점 선택 <input type="radio"> (1점~5점), 리뷰 내용 <textarea>, 제출 <button> 을 작성하세요
  • 추가5각 입력 항목에 <label> 을 붙여서 이름을 표시하세요

Q04
🍽️ 나만의 레시피 페이지
저장 파일명 → recipe.html
<h1><h2><ol><ul><table> <div><span><img><a><hr> + <header><main><section><footer> + <form><input><select><button>
20
점
📋 문제 상황
나만의 레시피 페이지를 만들어요. 기존 조건을 완성한 후 시맨틱 태그로 구조를 잡고 인분 수를 선택하는 간단한 폼을 추가하세요.
📌 기존 조건
  • 1<h1> 로 요리 이름, <h2> 로 "재료", "조리 순서", "주의사항" 소제목 3개를 작성하세요
  • 2<div> 로 "재료 영역"과 "조리 순서 영역"을 각각 묶으세요
  • 3<img> 로 완성된 요리 사진을 넣으세요 (alt 속성 필수)
  • 4<table> 로 재료명과 양을 표로 정리하세요 (thead, tbody, th, td 모두 사용)
  • 5<ol> 로 조리 순서를 4단계 이상 작성하세요
  • 6조리 시간/온도에 <span> 을 사용하세요
  • 7<ul> 로 주의사항 2가지 이상 작성하세요
  • 8<a> 로 유튜브 링크를 새 탭으로 연결하세요
  • 9소제목 사이에 <hr> 구분선을 넣으세요
✨ 추가 조건 — 시맨틱 구조 + 인분 선택 폼 추가
  • 추가1전체를 <header>, <main>, <footer> 로 나누세요
  • 추가2<main> 안의 재료 / 조리 순서 / 주의사항 각 영역을 <section> 으로 나누세요
  • 추가3재료 섹션 맨 위에 "🍽️ 인분 선택" <form> 을 추가하세요
  • 추가4폼 안에 <label> + <select> 로 1인분 / 2인분 / 4인분 선택을 만들고 <button> 적용 버튼을 넣으세요
  • 추가5<footer> 에 작성자 이름과 날짜를 넣으세요

Q05
🌟 나의 미니 홈페이지 — 랜딩 페이지
저장 파일명 → index.html
<h1><h2><p><ul><li><a><img> + <header><nav><main><section><footer> + <form><input><textarea><select><button>
20
점
📋 문제 상황
1번~4번에서 만든 4개의 페이지를 연결하는 메인 랜딩 페이지를 완성해요. 시맨틱 태그로 전체 구조를 잡고 방문자 메모 폼도 추가하세요. 나중에 CSS 수업 때 이 페이지를 예쁘게 꾸밀 거예요!
📌 기존 조건
  • 1<h1> 로 "나의 이름 + 미니 홈페이지" 대제목을 작성하세요
  • 2<img> 로 프로필 이미지를 넣으세요 (alt 속성 필수)
  • 3<p> 로 자기소개 한 줄을 작성하세요. <strong> 과 <em> 을 각각 1번 이상 사용하세요
  • 4<h2> 로 "📂 나의 작품 목록" 소제목을 작성하세요
  • 5<ul> + <li> + <a href="파일명"> 으로 4개 페이지를 링크 목록으로 연결하세요
  • 6각 링크에 이모지를 포함하세요 (예: 🐾 동물 소개 페이지)
  • 7<div> 로 "프로필 영역"과 "작품 목록 영역"을 각각 묶으세요
  • 8프로필과 작품 목록 사이에 <hr> 구분선을 넣으세요
✨ 추가 조건 — 시맨틱 전체 구조 + 방문자 메모 폼 추가
  • 추가1전체 페이지를 <header>, <nav>, <main>, <footer> 로 구분하세요
  • 추가2<header> 에 <h1> 과 프로필 <img> 를 넣으세요
  • 추가3<nav> 에 <ul> + <li> + <a> 로 4개 페이지 링크 메뉴를 넣으세요
  • 추가4<main> 안을 <section> 2개로 나누세요 — 첫 번째는 자기소개, 두 번째는 작품 목록
  • 추가5작품 목록 섹션 아래에 "💬 방문자 메모" <h2> 를 추가하고 <form> 을 작성하세요
    — <input type="text"> 이름 입력
    — <select> 로 방문 경로 선택 (직접방문/검색/추천)
    — <textarea> 로 메모 입력
    — <button type="submit"> 제출 버튼
  • 추가6<footer> 에 이름, 수업명, 날짜를 넣으세요
