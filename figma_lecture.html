Chapter 01

피그마, 세상을 뒤흔든 이야기

스타트업에서 시작해 디자인 업계를 완전히 바꾼 피그마의 드라마틱한 역사

🎯 피그마란?
Figma는 브라우저에서 바로 실행되는 협업형 UI/UX 디자인 툴입니다. 설치 없이 웹에서 열리고, 여러 명이 동시에 같은 화면을 편집할 수 있어 "구글 독스의 디자인 버전"이라 불립니다.
2012
🌱 아이디어의 탄생
Dylan Field(딜런 필드)와 Evan Wallace(에반 월리스)는 Brown University(브라운 대학교) 재학 중 아이디어를 구상합니다. Dylan은 Peter Thiel Fellowship에서 10만 달러 장학금을 받아 대학을 중퇴하고 본격적인 창업에 뛰어듭니다. 당시 목표는 단순했습니다 — "디자인 툴을 브라우저 위에서 실행하자."
2012~2016
🔬 4년간의 은밀한 개발
피그마 팀은 무려 4년간 스텔스 모드로 제품을 개발합니다. 브라우저에서 복잡한 벡터 그래픽을 실행하는 것은 당시 기술적으로 매우 어려운 도전이었습니다. WebGL과 WebAssembly 기술을 활용하여 네이티브 앱 수준의 성능을 브라우저에서 구현하는 데 집중했습니다.
2016년 9월
🚀 세상에 공개 — 디자인 업계 충격
Figma 1.0 공개 베타 출시. 당시 업계 표준이었던 Adobe Illustrator, Sketch에 익숙했던 디자이너들은 처음에 반신반의했습니다. "브라우저에서 그게 돼?" 하지만 한 번 써본 사람들은 돌아가지 않았습니다. 실시간 협업이라는 킬러 기능이 특히 주목받았습니다.
2019
🎨 FigJam 개발 시작 · 폭발적 성장
COVID-19 팬데믹 이후 원격 근무가 폭발적으로 증가하며 Figma의 성장도 가속화됩니다. "화이트보드도 협업으로!" 라는 개념 하에 FigJam 개발이 시작됩니다. 시리즈 D 투자로 기업 가치 20억 달러(약 2.6조원)를 달성합니다.
2021년 4월
💬 FigJam 출시
온라인 화이트보드 협업 툴 FigJam이 공식 출시됩니다. 스티커 메모, 투표 기능, 타이머 등 팀 회의에 특화된 기능들이 탑재되어 Miro, Mural 등의 경쟁사를 빠르게 추격합니다.
2022년 9월
💥 어도비의 200억 달러 인수 시도 — 세기의 빅딜
Adobe가 무려 200억 달러(약 26조원)에 Figma 인수를 발표합니다. 당시 Figma 연매출의 약 50배, 역대 소프트웨어 기업 인수 중 최고가 중 하나였습니다. 디자인 업계는 발칵 뒤집혔습니다.

😱 디자이너들의 반응: SNS에서 "#SaveFigma" 해시태그가 폭발했습니다. 많은 디자이너들이 "어도비가 피그마를 망칠 것"이라며 걱정했고, 대체 툴 목록이 공유되었습니다. Penpot, Sketch, Adobe XD 등의 대안 툴 다운로드 수가 급증했습니다.

2023년 12월
🚫 인수 무산 — EU·미국 규제 당국 제동
미국 법무부(DOJ)와 유럽연합(EU) 규제 당국이 독점 우려를 이유로 인수 반대 입장을 표명합니다. Adobe와 Figma는 결국 인수를 자진 철회합니다. Adobe는 Figma에 10억 달러(약 1.3조원)의 위약금을 지급해야 했습니다.

🎉 디자이너 커뮤니티의 반응: 많은 디자이너들이 환호했습니다. Figma는 오히려 이 사건을 계기로 독립성을 더욱 강조하게 됩니다. "어도비도 갖지 못한 것" 이미지가 오히려 브랜드 가치를 높였다는 분석도 있습니다.

2024
🤖 AI 시대 진입 — Config 2024
Figma AI 기능들이 대거 공개됩니다. 텍스트로 UI를 만드는 Make Designs, 더미 텍스트를 실제 콘텐츠로 바꾸는 Fill with AI, 자동 주석 생성 등 AI 기능이 핵심으로 자리잡습니다.
2025
🌐 Figma Sites & Draw — 영역 확장
Figma가 웹사이트 퍼블리싱 기능 Figma Sites를 출시하며 디자인-개발 간 경계를 더욱 허뭅니다. 현재 전 세계 디자인 팀의 70% 이상이 사용하는 업계 표준 툴로 자리매김했습니다. 사용자 수 400만 명 이상, Fortune 500 기업의 90%가 고객입니다.
Chapter 02

핵심 기본 툴 빠르게 마스터

처음 접하는 분들을 위한 필수 툴 간략 정리 — 이것만 알면 시작할 수 있습니다

💡 Figma 시작 방법: figma.com에서 무료 계정 생성 → 브라우저에서 바로 시작! 설치 불필요. (데스크탑 앱도 있어요)
🖱️

Select (선택 툴)

오브젝트를 클릭하거나 드래그하여 선택합니다. 다중 선택은 Shift+클릭, 모든 선택은 Ctrl+A입니다.

V
⬛

Frame (프레임)

웹/앱 화면의 기본 캔버스입니다. iPhone, Web, Desktop 등 기기별 프리셋이 있어 클릭 한 번으로 적용됩니다.

F
📝

Text (텍스트)

T키를 누르고 클릭하면 텍스트 입력 모드. 폰트, 크기, 굵기를 우측 패널에서 조정합니다. 한글 폰트도 완벽 지원!

T
⬜

Rectangle / Shape (도형)

사각형, 원, 다각형 등 기본 도형. Shift를 누르며 그리면 정사각형/정원이 됩니다. 모서리 반경으로 버튼 모양 만들기에 필수!

R / O / L
✏️

Pen (펜 툴)

벡터 드로잉 툴. 로고나 아이콘 제작에 사용합니다. 초보자는 처음에 잘 안 쓰지만 익숙해지면 강력합니다.

P
🧩

Component (컴포넌트)

반복 사용할 UI 요소를 컴포넌트로 만들면 하나를 수정할 때 연결된 모든 곳이 자동 업데이트 됩니다. 버튼, 카드, 헤더 등에 활용!

Ctrl+Alt+K
📐

Auto Layout (오토 레이아웃)

Flexbox처럼 작동하는 자동 정렬 기능. 내용이 늘어나도 레이아웃이 자동으로 조정됩니다. 반응형 UI 만들기의 핵심!

Shift+A
🔗

Prototype (프로토타입)

화면과 화면을 연결하여 클릭 가능한 시뮬레이션을 만듭니다. 링크를 공유하면 개발자나 클라이언트가 실제처럼 체험할 수 있습니다.

우측 패널 Prototype 탭

⌨️ 꼭 외워야 할 단축키 TOP 10

  • Ctrl+Z : 실행 취소 (가장 많이 쓸 거예요!)
  • Ctrl+D : 복제
  • Ctrl+G : 그룹화
  • Ctrl+Shift+G : 그룹 해제
  • Ctrl+[ / ] : 레이어 순서 앞/뒤로
  • Space+드래그 : 캔버스 이동
  • Ctrl+0 : 전체 화면에 맞추기
  • Ctrl+Shift+H/V : 수평/수직 중앙 정렬
  • Alt+드래그 : 간격 측정
  • Ctrl+/ : 명령어 검색 (모든 기능 빠른 접근)
Chapter 03

AI로 10배 빠르게 디자인하기

Figma에 내장된 AI 기능부터 외부 AI 툴 연동까지 — 지금 당장 쓸 수 있는 방법들

● Figma AI 내장

Make Designs — 텍스트 한 줄로 UI 생성

Figma 캔버스에서 AI에게 원하는 화면을 말로 설명하면 자동으로 UI를 만들어줍니다. 2024년 Config에서 발표된 가장 혁신적인 기능입니다.

  • "쇼핑몰 상품 카드 UI 만들어줘" → 자동 생성
  • "모바일 로그인 화면 3가지 버전으로" → 즉시 시안 3개
  • "다크 모드 버전으로 바꿔줘" → 자동 변환
  • 생성된 디자인을 베이스로 직접 수정 가능
Figma AI · Make Designs
▸ 프롬프트 입력
"카페 앱의 주문 화면을 만들어줘.
상품 이미지, 이름, 가격, 수량 선택,
장바구니 버튼이 있어야 해.
미니멀하고 따뜻한 느낌으로."

▸ AI 응답
✦ 프레임 생성 중...
✦ 컴포넌트 배치 중...
✦ 색상 팔레트 적용 중...
✓ 3개의 디자인 시안 생성 완료!
● Figma AI 내장

Fill with AI — 더미 데이터 끝!

Lorem ipsum 텍스트, 회색 플레이스홀더 이미지 대신 실제 같은 콘텐츠를 AI가 채워줍니다. 클라이언트 프레젠테이션이 훨씬 설득력 있어집니다.

  • 프레임에 "한국 카페 이미지 채워줘" → 즉시 이미지 삽입
  • 텍스트 레이어 선택 후 "사용자 리뷰 5개 생성" → 자연스러운 텍스트
  • 이름, 날짜, 가격 등 데이터 자동 생성
  • 다국어 텍스트 자동 번역·채우기
Figma AI · Fill with AI
▸ Before (기존 방식)
이름: Lorem Ipsum
직업: Lorem dolor sit
리뷰: Lorem ipsum...

▸ After (AI Fill 적용)
이름: 김서연 (28세, 서울)
직업: UX 디자이너
리뷰: "정말 편리해요! 주문하는 데 30초도 안 걸렸어요 ☕"
● 외부 AI 연동

ChatGPT / Claude로 디자인 기획하기

Figma 작업 전에 AI와 먼저 대화하면 훨씬 빠르게 시작할 수 있습니다. 아이디어 구체화부터 컴포넌트 설계까지 AI를 기획 파트너로 활용하세요.

  • 와이어프레임 설계 아이디어 요청
  • 색상 팔레트 추천 (브랜드 키워드 입력)
  • UX 카피라이팅 초안 생성
  • 사용자 페르소나 설계 도움받기
  • 접근성(Accessibility) 체크리스트 생성
ChatGPT · 디자인 기획 보조
▸ 프롬프트 예시
"반려동물 용품 쇼핑몰 앱의
메인 화면에 필요한 UI 컴포넌트 목록과
각각의 역할을 정리해줘.
타겟은 20~30대 펫 오너야."

▸ AI 답변 → Figma에 그대로 적용!

● Figma AI 내장

Rename Layers & Auto Annotate

개발자와의 협업을 AI가 도와줍니다. 레이어 이름을 자동으로 정리해주고, 디자인 요소에 자동으로 주석(Annotation)을 달아줍니다.

  • "btn_primary_hover" 같은 의미있는 레이어명 자동 부여
  • 개발 핸드오프용 스펙 주석 자동 생성
  • 색상, 폰트, 간격 값 자동 표시
  • 접근성 레이블(aria-label) 자동 제안

Chapter 04

플러그인 완전 정복

Figma의 진짜 힘은 플러그인에 있습니다. 커뮤니티에서 무료로 설치할 수 있는 강력한 플러그인들을 카테고리별로 소개합니다.

💡 플러그인 설치 방법: 메인 메뉴 → Plugins → Browse plugins in Community → 원하는 플러그인 검색 → Install 클릭! 설치 후 우클릭 → Plugins에서 바로 사용 가능합니다.

AI 기반 텍스트 생성, 아이콘 생성, 이미지 생성이 모두 가능한 올인원 AI 플러그인. GPT-4 기반으로 카피라이팅부터 아이콘 디자인까지 해결합니다.

FigJam — 협업의 마법

팀이 함께 아이디어를 내고, 정리하고, 결정하는 온라인 화이트보드. AI까지 탑재됐습니다.

FigJam이란?

FigJam은 Figma 팀이 2021년 출시한 온라인 협업 화이트보드입니다. 브레인스토밍, 회고 미팅, 사용자 스토리 맵핑, 플로우차트 작성 등 팀이 함께 생각을 정리하는 모든 작업에 활용할 수 있습니다.

Figma와 같은 플랫폼에 있어서 디자인 파일과 기획 보드를 오가며 작업할 수 있고, 팀원들이 동시에 접속하여 실시간으로 아이디어를 붙이고 정리합니다.

🤖 FigJam AI 완전 정복

FigJam에 내장된 AI 기능으로 회의를 더 효율적으로

📝 회의 노트 자동 정리
산발적인 스티커 메모들을 AI가 주제별로 자동 분류하고 요약합니다. "아이디어 모아줘" 한 마디로 30분 회의 내용이 깔끔하게 정리됩니다.
💡 아이디어 확장
스티커 메모 하나를 선택하고 "더 많은 아이디어 생성해줘"라고 하면 AI가 관련 아이디어를 추가로 제안합니다.
✅ Action Items 자동 생성
회의 종료 후 "할 일 목록 만들어줘"라고 하면 논의된 내용에서 실행 가능한 태스크를 자동으로 추출합니다.
🗺️ 마인드맵 자동 생성
핵심 키워드 하나를 입력하면 관련 개념들이 가지처럼 펼쳐지는 마인드맵을 자동으로 생성합니다.

Figma vs FigJam 비교표

어떤 상황에 무엇을 써야 할까요?

기능/특징 Figma FigJam
주요 용도 UI/UX 디자인, 프로토타입 브레인스토밍, 기획, 회의
대상 디자이너 중심 팀 전체 (기획자, 개발자 등)
벡터 편집 ✓ 완전 지원 ✗ 미지원
스티커 메모 ✗ 제한적 ✓ 핵심 기능
투표 기능 ✗ 없음 ✓ 내장
컴포넌트 시스템 ✓ 강력 지원 ✗ 없음
AI 기능 ✓ Make Designs, Fill ✓ 정리, 마인드맵
무료 플랜 ✓ 3프로젝트 무료 ✓ 무료 사용 가능
추천 상황 디자인 시안 제작 시 킥오프 미팅, 기획 워크샵

🎓 초보자를 위한 학습 로드맵

  • Week 1 : Figma 계정 생성 → 기본 툴 익히기 (Frame, 도형, 텍스트)
  • Week 2 : Component와 Auto Layout 마스터 → 간단한 버튼/카드 만들기
  • Week 3 : FigJam으로 팀 협업 경험 → 브레인스토밍 보드 만들기
  • Week 4 : AI 기능 활용 (Make Designs, Fill with AI) → 자동화 경험
  • Week 5 : 플러그인 5개 설치 및 활용 (Unsplash, Iconify, Magician 등)
  • Week 6 : 첫 번째 앱 UI 프로젝트 완성 → 프로토타입 링크 공유
