Q01
📅 우리 반 주간 시간표
저장 파일명 → timetable.html
★ rowspan ★ colspan thead / tbody th / td caption strong / mark
30
점
📋 문제 상황
월요일부터 금요일까지 1교시~5교시 주간 시간표를 HTML 표로 만들어요. 같은 과목이 연속으로 이어지는 경우 rowspan 으로 세로 병합하고, 점심시간은 colspan 으로 전체 요일을 가로 병합해서 표현하세요.
🔴 핵심 포인트
rowspan : 같은 과목이 연달아 2교시 이상일 경우 → 세로로 합치기
colspan : 점심시간처럼 모든 칸이 하나로 묶일 경우 → 가로로 합치기
병합한 만큼 다른 행의 <td> 를 빼줘야 칸이 어긋나지 않아요!
📌 작성 조건
  • 1<h1> 로 "우리 반 주간 시간표" 제목을 작성하세요
  • 2<caption> 으로 표 제목 "2025년 1학기 시간표" 를 넣으세요
  • 3<thead> 에 <th> 로 교시 / 월 / 화 / 수 / 목 / 금 헤더를 작성하세요
  • 4<tbody> 에 1교시~5교시 시간표를 자유롭게 작성하세요 (과목은 마음대로)
  • 5🔴 심화 월요일 또는 어느 요일이든 같은 과목이 연달아 2교시 이상인 경우를 만들어서 rowspan="2" 이상으로 세로 병합하세요 (최소 1곳 이상)
  • 6🔴 심화 3교시와 4교시 사이에 점심시간 행을 추가하고 colspan="6" 으로 교시 포함 전체 6칸을 하나로 합치세요
  • 7가장 좋아하는 과목에 <mark> 강조를, 가장 싫어하는 과목에 <em> 을 적용하세요
  • 8표 아래에 <small> 로 "※ 시간표는 변경될 수 있습니다" 주의사항을 작성하세요

💡 rowspan 작성 힌트!
<td rowspan="2">체육</td> 을 쓴 행에서는 다음 행에 같은 위치의 <td> 를 하나 빼야 해요.
1행에 rowspan="2" 를 쓰면 → 2행에서 그 칸에 해당하는 td 1개 삭제!
Q02
🎬 이번 주 박스오피스 TOP 5
저장 파일명 → boxoffice.html
★ colspan (대제목) ★ tfoot (합계) thead / tbody caption a / img ol / li strong / del / mark
35
점
📋 문제 상황
이번 주 영화 박스오피스 TOP 5 순위표를 만들어요. 표 상단에 colspan 으로 큰 제목 행을 만들고, tfoot 에 전체 누적 관객수 합계를 넣고, 관련 링크와 이미지도 포함하세요.
🔴 핵심 포인트
colspan 대제목 : 표 맨 위에 전체 열을 합친 제목 행을 만들어요
<thead> 안에 첫 번째 <tr> 은 colspan 으로 합친 대제목, 두 번째 <tr> 은 각 열의 세부 제목이에요
tfoot : 표 맨 아래 합계나 요약 정보를 넣는 영역이에요
📌 작성 조건
  • 1<h1> 로 "🎬 이번 주 박스오피스 TOP 5" 제목, <p> 로 기준 날짜를 작성하세요
  • 2<img> 로 영화 관련 이미지를 1개 이상 넣으세요 (alt 속성 필수)
  • 3🔴 심화 <thead> 첫 번째 행에 colspan 으로 전체 열을 합친 "2025년 1월 4주차 박스오피스" 대제목 행을 만드세요
  • 4<thead> 두 번째 행에 <th> 로 순위 / 영화제목 / 장르 / 누적관객 / 전주대비 / 링크 세부 헤더를 작성하세요
  • 5<tbody> 에 영화 5편의 정보를 행으로 작성하세요 (영화는 실제 또는 가상으로 자유롭게)
  • 61위 영화 제목에 <mark> + <strong>, 지난주보다 순위가 내려간 영화에 <del> 을 사용하세요
  • 7각 영화 링크 칸에 <a> 로 예매 링크를 새 탭으로 연결하세요
  • 8🔴 심화 <tfoot> 에 "TOP 5 합계" 행을 추가하고, 영화제목~전주대비 칸은 colspan="4" 로 합쳐서 총 누적관객 합계를 표시하세요
  • 9표 아래에 <small> 로 출처를 표시하고, <ol> 로 개인 추천 TOP 3 목록을 작성하세요

💡 tfoot colspan 힌트!
tfoot 행에서 "합계" 칸 1개 + 중간 내용 colspan 합친 칸 + 숫자 칸 = 총 6칸이 돼야 해요.

Q03
🌟 나의 HTML 작품 포트폴리오 페이지
저장 파일명 → index.html
★ colspan + rowspan 혼합 thead / tbody / tfoot caption h1~h3 / p / ul / ol a / img / div / span strong / em / mark / del hr / small
35
점
📋 문제 상황
지금까지 만든 모든 HTML 파일을 연결하는 포트폴리오 메인 페이지를 완성해요. 작품 목록을 colspan + rowspan 을 모두 사용한 복잡한 표로 정리하고 자기소개, 작품 목록, 배운 태그 정리까지 한 페이지에 모두 담으세요.
🔴 핵심 포인트 — colspan + rowspan 동시 사용!
이번 문제는 colspan 과 rowspan 을 같은 표 안에서 둘 다 써야 해요.
작품 목록 표에서 카테고리(기초/심화)를 rowspan 으로 묶고, 표 맨 위 대제목은 colspan 으로 전체를 합쳐요.
📌 작성 조건
  • 1<h1> 로 "나의 이름 + HTML 포트폴리오", <img> 로 프로필 이미지, <p> 로 자기소개 한 줄을 작성하세요. <strong> 과 <em> 을 각각 1번 이상 사용하세요
  • 2<h2> 로 "📂 작품 목록", "🛠️ 배운 태그", "🔗 유용한 링크" 소제목 3개를 작성하세요
  • 3🔴 심화 작품 목록 <table> 에서 <thead> 첫 번째 행을 colspan 으로 전체 합친 대제목 행으로 만드세요
  • 4🔴 심화 <tbody> 에서 "기초 과정" 과 "심화 과정" 카테고리 칸을 각각 rowspan 으로 세로 병합하세요
    기초 4개 작품 → rowspan="4" / 심화 2개 작품 → rowspan="2"
  • 5각 작품 행에 작품명, 설명, 링크 (<a href="파일명">) 를 넣고 아래 6개 파일을 모두 연결하세요
    animal.html / book.html / movie.html / recipe.html / timetable.html / boxoffice.html
  • 6🔴 심화 <tfoot> 에 "총 작품 수" 행을 추가하고 colspan 으로 적절히 병합해서 총 6개 작품이라고 표시하세요
  • 7<h2> 배운 태그 섹션에 <ul> 로 카테고리별 태그 목록을 작성하고 <mark> 로 가장 어려웠던 태그를 강조하세요
  • 8<h2> 유용한 링크 섹션에 <ol> 로 추천 사이트 목록을 만들고 <a> 로 모두 새 탭 연결하세요
  • 9페이지 맨 아래 <hr> 을 넣고 <p> 에 이름, 수업명, 날짜를 쓴 뒤 <small> 로 저작권 표시를 하세요
