📌 작성 안내
① 모든 문제는 HTML 태그만 사용해요 (CSS 사용 불가)
② 반드시 <!DOCTYPE html> 부터 시작하는 완전한 HTML 문서 구조로 작성해요
③ 태그는 모두 소문자로, 여는 태그와 닫는 태그를 정확하게 써요
④ 각 문제의 파일명을 정확하게 지켜서 저장해야 5번 문제에서 연결이 돼요!
📁 파일 저장 규칙 — 모든 파일은 같은 폴더에 저장해요!
📁 내 이름 폴더
├── animal.html ← 1번 문제
├── book.html ← 2번 문제
├── movie.html ← 3번 문제
├── recipe.html ← 4번 문제
└── index.html ← 5번 문제 (랜딩 페이지)
Q01
🐾 내가 좋아하는 동물 소개 페이지
저장 파일명 → animal.html
<h1><h2> <p><br><hr> <strong><em><mark> <div> <span> <img> <ul><li>
20
점
📋 문제 상황
내가 가장 좋아하는 동물을 친구들에게 소개하는 페이지를 만들어요. 동물의 특징, 사는 곳, 먹이를 정리하고 이미지도 넣어보세요.
📌 작성 조건
  • 1<h1> 로 동물 이름을 대제목으로 작성하세요
  • 2<h2> 로 "특징", "사는 곳", "먹이" 소제목 3개를 작성하세요
  • 3<div> 로 각 소제목 영역을 묶으세요
  • 4<img> 로 동물 이미지를 넣으세요 (alt 속성 필수)
  • 5<ul> + <li> 로 특징 3가지 이상 목록으로 작성하세요
  • 6특징 중 핵심 단어에 <strong>, 감탄 표현에 <em>, 중요 내용에 <mark> 를 사용하세요
  • 7동물의 색깔이나 크기를 설명하는 부분에 <span> 을 사용하세요
  • 8소제목 사이에 <hr> 구분선을 넣으세요

Q02
📚 나의 독서 목록 페이지
저장 파일명 → book.html
<h1><h2> <ul><ol><li> <strong><del><mark> <div> <img> <a> <p>
20
점
📋 문제 상황
올해 읽은 책과 읽고 싶은 책 목록 페이지를 만들어요. 이미 읽은 책은 취소선으로 표시하고, 책 검색 사이트 링크도 넣어보세요.
📌 작성 조건
  • 1<h1> 로 "나의 독서 목록 📚" 제목을 작성하세요
  • 2<h2> 로 "읽고 싶은 책", "이미 읽은 책" 소제목 2개를 작성하세요
  • 3<div> 로 두 영역을 각각 묶으세요
  • 4<ol> 로 읽고 싶은 책 3권 이상 순위 목록으로 작성하고 1순위 책에 <mark> 와 <strong> 을 사용하세요
  • 5<ul> 로 읽은 책 2권 이상 작성하고 모든 책에 <del> 취소선을 적용하세요
  • 6<img> 로 책 표지 이미지를 1개 이상 넣으세요 (alt 속성 필수)
  • 7<a> 로 책 검색 사이트 링크를 넣고 target="_blank" 로 새 탭에서 열리게 하세요

Q03
🎬 영화 리뷰 페이지
저장 파일명 → movie.html
<h1><h2><h3> <p><ul><hr> <strong><em><small> <div><span> <img> <a>
20
점
📋 문제 상황
최근에 본 영화 한 편을 리뷰하는 페이지를 만들어요. 포스터 이미지, 영화 정보, 줄거리, 평점을 모두 포함해서 작성해 보세요.
📌 작성 조건
  • 1<h1> 로 영화 제목, <h2> 로 "기본정보" "줄거리" "나의 평점" 소제목 3개를 작성하세요
  • 2<div> 로 영화 정보 카드 전체를 묶으세요
  • 3<img> 로 영화 포스터 이미지를 넣으세요 (alt 속성 필수)
  • 4<ul> + <li> 로 감독, 장르, 개봉년도 기본 정보를 목록으로 작성하세요
  • 5줄거리 <p> 안에서 핵심 내용을 <strong>, 느낌을 <em> 으로 표현하세요
  • 6평점 별 이모지 ⭐⭐⭐⭐☆ 에 <span> 을 사용하세요
  • 7<small> 로 출처를 작게 표시하고, <a> 로 예매 사이트 링크를 새 탭으로 연결하세요
  • 8소제목 사이에 <hr> 구분선을 넣으세요

Q04
🍽️ 나만의 레시피 페이지
저장 파일명 → recipe.html
<h1><h2><h3> <ol><ul><li> <table><thead><tbody><tr><th><td> <div><span> <img><a><hr>
20
점
📋 문제 상황
내가 좋아하는 음식의 레시피 페이지를 만들어요. 재료는 표로, 조리 순서는 ol 로 정리하고 이미지와 유튜브 링크도 넣어보세요.
📌 작성 조건
  • 1<h1> 로 요리 이름, <h2> 로 "재료", "조리 순서", "주의사항" 소제목 3개를 작성하세요
  • 2<div> 로 "재료 영역"과 "조리 순서 영역"을 각각 묶으세요
  • 3<img> 로 완성된 요리 사진을 넣으세요 (alt 속성 필수)
  • 4<table> 로 재료명과 양을 표로 정리하세요 (thead, tbody, th, td 모두 사용)
  • 5<ol> 로 조리 순서를 4단계 이상 작성하세요
  • 6조리 시간이나 온도를 표시하는 부분에 <span> 을 사용하세요
  • 7<ul> 로 주의사항 2가지 이상 작성하세요
  • 8<a> 로 유튜브 레시피 링크를 새 탭으로 연결하세요
  • 9소제목 사이에 <hr> 구분선을 넣으세요

Q05
🌟 나의 미니 홈페이지 — 랜딩 페이지 만들기
저장 파일명 → index.html
<h1><h2><p> <div> <ul><li> <a> <img> <strong><em> <hr>
20
점
📋 문제 상황
1번~4번에서 만든 4개의 페이지를 하나로 연결하는 나만의 메인 페이지를 만들어요.
이 페이지가 바로 내 미니 홈페이지의 첫 화면(랜딩 페이지)이에요.
나중에 CSS 수업 때 이 페이지를 예쁘게 꾸밀 거예요! 지금은 구조만 완성해요.
📌 작성 조건
  • 1<h1> 로 "나의 이름 + 미니 홈페이지" 대제목을 작성하세요
  • 2<img> 로 나를 소개하는 프로필 이미지를 넣으세요 (alt 속성 필수)
  • 3<p> 로 나를 소개하는 한 줄 소개글을 작성하세요. <strong> 과 <em> 을 각각 1번 이상 사용하세요
  • 4<h2> 로 "📂 나의 작품 목록" 소제목을 작성하세요
  • 5<ul> + <li> 로 4개의 페이지 목록을 만들고 각 항목에 <a href="파일명"> 으로 링크를 연결하세요
  • 6각 링크 텍스트에 이모지를 포함해서 작성하세요 (예: 🐾 동물 소개 페이지)
  • 7<div> 로 "프로필 영역"과 "작품 목록 영역"을 각각 묶으세요
  • 8프로필과 작품 목록 사이에 <hr> 구분선을 넣으세요
🖥️ 결과 화면 예시
브라우저 화면 (index.html) — 카드를 클릭하면 각 페이지로 이동해요!
