Q01
🐾 동물 소개 페이지에 font · text 스타일 적용하기
수정 파일 → animal.html   새로 만들 파일 → css/animal.css
선택자 color font-family font-size font-weight line-height text-align text-decoration
20
점
📋 문제 상황
HTML 수업에서 만든 animal.html에 CSS 파일을 연결하고, 선택자를 활용해서 글자 스타일을 꾸며보세요.
태그 선택자, 클래스 선택자, id 선택자를 모두 한 번 이상 사용해야 합니다.
📌 기본 조건
  • 1 animal.html 의 <head> 안에 <link rel="stylesheet" href="css/animal.css"> 를 추가해서 외부 CSS 파일을 연결하세요
  • 2 태그 선택자로 body 에 font-family를 설정하세요 — 구글 폰트 또는 기본 sans-serif 계열 사용
  • 3 태그 선택자로 h1 의 color를 원하는 색으로 지정하고, text-align: center를 적용하세요
  • 4 태그 선택자로 p 에 line-height: 1.8 과 font-size: 15px 를 설정하세요
  • 5 동물 이름이 있는 태그 (예: h2) 에 color와 font-weight: 700을 설정하세요
  • 6 a 태그에 text-decoration: none을 적용해서 밑줄을 없애세요
✨ 추가 조건 — class · id 선택자 활용
  • 추가1 동물 이름 태그에 class="animal-name"을 추가하고, 클래스 선택자로 font-size와 color를 스타일하세요
  • 추가2 페이지 제목이 있는 <h1> 을 감싼 요소에 id="main-title"을 추가하고, id 선택자로 background-color와 padding을 설정하세요
  • 추가3 동일한 스타일을 여러 태그에 동시 적용하는 다중 태그 선택자를 1개 이상 사용하세요
    • 예시 : h1, h2 { color: #333; }
💡 CSS 파일 연결 순서 기억하기
외부 CSS 링크는 <head> 안에서 다른 내용보다 아래쪽에 배치하세요.
구글 폰트를 쓴다면 <link> 폰트 연결을 먼저 쓰고, 그 아래에 animal.css 연결을 작성합니다.
🖥️ CSS 작성 예시 (참고용)
/* css/animal.css */ body { font-family: 'Noto Sans KR', sans-serif; line-height: 1.6; } h1 { color: #1a5fa8; text-align: center; } .animal-name { font-size: 22px; font-weight: 700; color: #c8390a; }
  • CSS 파일을 별도로 만들고 <link> 로 연결했나요?
  • 태그 선택자를 사용했나요? (body, h1, p, a 등)
  • 클래스 선택자를 사용했나요? (.클래스이름)
  • id 선택자를 사용했나요? (#아이디이름)
  • color 값을 헥사코드 또는 rgb() 로 표기했나요?
Q02
📚 독서 목록 페이지에 text 스타일 심화 적용하기
수정 파일 → book.html   새로 만들 파일 → css/book.css
font 단축속성 text-indent letter-spacing text-transform text-decoration font-style
20
점
📋 문제 상황
book.html 에 CSS를 연결하고, text 관련 속성들을 활용해 가독성 좋은 독서 목록 페이지로 꾸며보세요.
font 단축 속성도 반드시 1회 이상 사용해야 합니다.
📌 기본 조건
  • 1 book.html 에 css/book.css 를 외부 연결하세요
  • 2 페이지 본문 p 태그에 text-indent: 20px 를 적용해 첫 줄 들여쓰기를 설정하세요
  • 3 책 제목이 있는 태그에 letter-spacing: 1px 로 자간을 설정하세요
  • 4 font 단축 속성을 사용해 굵기 + 크기/줄간격 + 서체를 한 번에 작성하세요
    예: font: bold 16px/1.8 'Noto Sans KR', sans-serif;
  • 5 읽은 책 목록에서 완독한 책에 클래스를 추가하고 text-decoration: line-through 로 취소선을 표시하세요
  • 6 책 장르(소설/에세이 등)가 영어로 표기된 경우 text-transform: uppercase 로 대문자 처리하세요
✨ 추가 조건 — font 단축 속성 완전 정복
  • 추가1 페이지 제목 h1에 font-style: italic 을 적용하고, 책 소개 문장 p에는 font-style: normal 로 해제하세요
  • 추가2 font 단축 속성에서 font-weight, font-style, font-variant 중 2가지 이상을 포함해서 작성하세요
    • 첫 번째 구간(굵기/스타일/variant)은 순서 변경 가능, 생략 가능
    • 두 번째 구간(크기/줄간격)에서 크기는 필수
    • 세 번째 구간(서체)은 절대 생략 불가
  • 추가3 body에 color를 rgba() 형식으로 작성해서 텍스트 색상을 지정하세요
    • 예: color: rgba(30, 30, 30, 0.85);
💡 font 단축 속성 순서 기억법
굵기·스타일·variant → 크기/줄간격 → 서체
크기와 서체는 반드시 있어야 해요. 줄간격은 기본값(1.4)이면 생략 가능해요.
🖥️ CSS 작성 예시 (참고용)
/* css/book.css */ /* font 단축 속성 사용 */ body { font: 400 15px/1.8 'Noto Sans KR', sans-serif; color: rgba(30, 30, 30, 0.85); } h1 { font: bold italic 28px 'Noto Sans KR', serif; letter-spacing: 2px; } .done { text-decoration: line-through; color: #aaa; } p { text-indent: 20px; }
  • font 단축 속성을 올바른 순서로 작성했나요?
  • font 단축 속성에 서체(font-family)를 마지막에 넣었나요?
  • text-indent는 블록 요소에 적용했나요?
  • 완독 책에 클래스를 추가하고 CSS로 취소선을 줬나요?
  • color를 rgba() 형식으로 작성했나요?
Q03
🎬 영화 리뷰 페이지에 background 속성 적용하기
수정 파일 → movie.html   새로 만들 파일 → css/movie.css
background-color background-image background-repeat background-size background-position + 스트라이프 패턴
20
점
📋 문제 상황
movie.html 에 CSS를 연결하고, background 관련 속성을 활용해 영화 리뷰 페이지의 배경을 꾸며보세요.
background-image를 이용한 스트라이프 패턴을 반드시 한 곳 이상 적용해야 합니다.
📌 기본 조건
  • 1 movie.html 에 css/movie.css 를 외부 연결하세요
  • 2 body 에 background-color 를 설정해 페이지 전체 배경색을 지정하세요 (흰색 제외)
  • 3 영화 제목이 담긴 영역(div 또는 header)에 background-color와 color를 설정해 눈에 띄게 꾸미세요
  • 4 페이지 어느 한 영역(예: 제목 배경, 리뷰 카드)에 수평 스트라이프 패턴을 적용하세요
    background-image: linear-gradient(...) + background-size 사용
  • 5 스트라이프 패턴을 적용한 영역에 background-color 도 함께 설정해 투명한 줄 사이에 배경색이 보이게 하세요
  • 6 별점이나 장르 태그 등 작은 요소에 background-color와 color를 조합해서 배지(badge) 스타일을 만드세요
✨ 추가 조건 — background 심화
  • 추가1 수평 스트라이프 외에 사선 스트라이프(45deg)도 한 곳에 추가로 적용하세요
    • background-image: linear-gradient(45deg, 색상 25%, transparent 25%), linear-gradient(45deg, transparent 75%, 색상 75%);
    • background-size: 20px 20px;
    • background-position: 0 0, 10px 10px; ← size의 절반값!
  • 추가2 background 단축 속성을 사용해서 color + image + repeat + position 을 한 줄로 작성하는 구간을 1개 이상 만드세요
  • 추가3 영화 포스터 이미지가 있다면 background-size: cover 와 background-position: center 를 활용해서 배경 이미지로 넣어보세요
💡 스트라이프 패턴 핵심 원리
수평 줄무늬 (레이어 1개): linear-gradient(0deg, 색상 50%, transparent 50%) + background-size: 20px 20px
사선 줄무늬 (레이어 2개): background-position 의 두 번째 레이어는 항상 size ÷ 2 만큼 이동시켜야 이음새가 맞아요!
🖥️ CSS 작성 예시 (참고용)
/* css/movie.css */ /* body 배경색 */ body { background-color: #1a1a2e; color: #eee; } /* 수평 스트라이프 */ .stripe-area { background-color: #ffffff; background-image: linear-gradient(0deg, #afa9ec 50%, transparent 50%); background-size: 20px 20px; } /* 장르 배지 */ .genre-badge { background-color: #c8390a; color: #fff; }
  • body에 background-color를 설정했나요?
  • linear-gradient로 스트라이프 패턴을 만들었나요?
  • background-size를 설정했나요?
  • 스트라이프 영역에 background-color도 함께 설정했나요?
  • 사선 스트라이프의 background-position 두 번째 값이 size/2인가요?
Q04
🍽️ 레시피 카드 페이지에 박스 모델 적용하기
수정 파일 → recipe.html   새로 만들 파일 → css/recipe.css
width / height padding border margin 단위(px · % · em)
20
점
📋 문제 상황
recipe.html 에 CSS를 연결하고, padding · border · margin 을 활용해서 레시피 카드처럼 보이도록 꾸며보세요.
박스 모델의 4방향 단축 속성을 올바르게 사용하는 것이 핵심입니다.
📌 기본 조건
  • 1 recipe.html 에 css/recipe.css 를 외부 연결하세요
  • 2 레시피 전체를 감싸는 div 에 width: 600px 와 margin: 0 auto 를 적용해서 가운데 정렬하세요
  • 3 레시피 제목 영역에 padding 을 단축 속성 2값으로 작성하세요 — 상하 / 좌우를 다르게 설정
  • 4 재료 목록 ul 에 border 를 설정하세요 — 선 굵기 / 선 색상 / 선 스타일(solid 또는 dashed) 모두 포함
  • 5 조리 순서 ol 각 li 에 padding 을 적용해서 항목 간 내부 여백을 만드세요
  • 6 h2, h3 등 소제목에 margin-bottom 을 설정해서 제목과 내용 사이 여백을 만드세요
✨ 추가 조건 — border 스타일 다양하게 + 단위 활용
  • 추가1 border 스타일을 최소 2가지 이상 다르게 사용하세요
    • solid (실선) / dashed (점선) / dotted (점점선) 중 선택
    • 예: 재료 목록은 dashed, 카드 전체는 solid
  • 추가2 padding 단축 속성 4가지 방법을 모두 사용해보세요 (값 1개 / 2개 / 3개 / 4개)
    • 값 1개: padding: 10px — 사방 동일
    • 값 2개: padding: 10px 20px — 상하 / 좌우
    • 값 3개: padding: 10px 20px 30px — 상 / 좌우 / 하
    • 값 4개: padding: 10px 20px 30px 40px — 시계방향 상·우·하·좌
  • 추가3 width를 px 이 아닌 % 단위로도 사용해 보세요 (예: width: 80%) 그리고 %와 px의 차이를 브라우저 창 크기를 바꿔가며 확인해 보세요
💡 padding vs margin 헷갈릴 때 기억법
padding = 박스 안쪽 여백 → background-color가 채워짐, 클릭 영역도 늘어남
margin = 박스 바깥 여백 → 투명한 공간, 클릭 안 됨
버튼을 크게 만들고 싶으면 padding을 쓰는 게 맞아요!
🖥️ CSS 작성 예시 (참고용)
/* css/recipe.css */ .recipe-card { width: 600px; margin: 0 auto; /* 상하:0 / 좌우:auto → 가운데 정렬 */ border: 2px solid #d46b00; padding: 20px 30px; /* 상하:20px / 좌우:30px */ } .ingredient-list { border: 1px dashed #ccc; padding: 10px 20px 10px 30px; margin-bottom: 16px; } h2 { margin-bottom: 12px; padding-bottom: 8px; border-bottom: 2px solid #d46b00; }
  • 레시피 카드를 감싸는 div에 width와 margin: 0 auto를 설정했나요?
  • padding 단축 속성 2값을 사용했나요? (상하 / 좌우)
  • border에 굵기·색상·스타일 3가지를 모두 넣었나요?
  • border 스타일을 2가지 이상 사용했나요?
  • padding의 4가지 작성법을 모두 시도했나요?
Q05
🌟 나의 미니 홈페이지 — CSS 종합 스타일링
수정 파일 → index.html   새로 만들 파일 → css/style.css
1~4번 내용 종합 font · text background padding · border · margin
20
점
📋 문제 상황
HTML 수업 마지막에 만든 index.html 랜딩 페이지에 배운 CSS를 모두 활용해서 나만의 미니 홈페이지를 완성하세요.
1~4번에서 배운 모든 내용을 골고루 사용해야 하며, 본인만의 스타일로 꾸미는 것이 목표입니다.
📌 필수 조건
  • 1 index.html 에 css/style.css 를 외부 연결하세요. 구글 폰트를 사용한다면 CSS 링크보다 위에 배치하세요
  • 2 header 영역에 background-color 또는 스트라이프 패턴을 적용하고, padding으로 내부 여백을 충분히 주세요
  • 3 nav 안의 링크들에 text-decoration: none 과 color를 설정해서 메뉴처럼 보이게 하세요
  • 4 main 안의 section 들에 padding과 margin-bottom을 적용해서 섹션 사이 여백을 만드세요
  • 5 작품 목록 ul li 에 border-bottom을 설정해서 리스트 구분선을 만드세요
  • 6 폼 요소 (input, select, textarea) 에 border와 padding을 적용하세요
  • 7 button 에 background-color, color, padding, border 를 적용해서 버튼 스타일을 만드세요
  • 8 footer 에 background-color, color, text-align: center, padding을 적용하세요
🎨 추가 조건 — 나만의 스타일 완성
  • 추가1 페이지 전체에서 사용하는 주요 색상 3가지를 정하고 일관성 있게 적용하세요
    (예: 포인트색 · 배경색 · 텍스트색)
  • 추가2 header 또는 nav 영역에 스트라이프 패턴을 적용하세요 (수평 또는 사선 자유 선택)
  • 추가3 구글 폰트를 연결해서 body 전체에 적용하고, 제목(h1)에는 본문과 다른 font-weight를 주세요
  • 추가4 지금까지 배운 모든 속성을 최소 1번 이상 사용했는지 아래 체크리스트로 확인하세요
    • color / font-family / font-size / font-weight / line-height
    • text-align / text-decoration / letter-spacing
    • background-color / background-image (linear-gradient)
    • background-size / background-position
    • width / padding / border / margin
💡 작업 순서 추천
① body 기본 스타일 (font, color, background) → ② header 꾸미기 → ③ nav 링크 스타일 → ④ main 섹션 여백 → ⑤ 폼 + 버튼 → ⑥ footer

🖥️ CSS 작성 예시 (참고용)
/* css/style.css — index.html 스타일 */ body { font-family: 'Noto Sans KR', sans-serif; background-color: #f5f3ee; color: #1a1a1a; line-height: 1.8; } header { background-color: #1a1a2e; color: #fff; padding: 30px 20px; text-align: center; } nav a { text-decoration: none; color: #1a5fa8; font-weight: 700; } button { background-color: #5a2d8a; color: #fff; border: none; padding: 10px 24px; } footer { background-color: #1a1a2e; color: rgba(255,255,255,0.6); text-align: center; padding: 20px; }
  • header에 background-color와 padding을 적용했나요?
  • nav a 에 text-decoration: none을 설정했나요?
  • section 에 padding과 margin을 설정했나요?
  • 버튼에 background-color, color, padding, border를 모두 설정했나요?
  • footer에 background-color, text-align: center, padding을 설정했나요?
  • 스트라이프 패턴을 한 곳 이상 적용했나요?
